Welcome to our community repository of Umbraco package ideas!

The concept behind this repository is to provide a better collaboration and discussion space for Umbraco package ideas.

Around 2012, I made some of the ideas public. I continued to add ideas to the list up until about 2015, when I moved the ideas over to a GitHub repository, with an intention to have a more fully featured website - although this didn't really happen. Fast forward to 2022, it's Hacktoberfest time, and the Umbraco community seems to have a renewed interest in Package Ideas. I decided to scrap the separate website idea, favouring to use GitHub features (that weren't around in 2015), like discussions and project boards.

Give an idea?

If you have an idea for a potential Umbraco package and would like to share it the world, please do feel free to open an issue on the GitHub repository. No problem if you don't have the coding skills, not enough hours in the day, or unsure where to start - by sharing your idea it will be open and available for the Umbraco community to collaborate on together.

Take an idea?

What if you have the coding skills but unsure what to develop... please do take an idea! As long as you credit where the original idea came from, you are free to take the idea and run with it. Although collaboration will always be encouraged, choosing to go solo is totally fine, (I often prefer to do most of my initial development work alone, then share later for validation and iteration). The execution of any idea is a big win for the Umbraco community!

Once you claim an idea for implementation, I'll move the issue over to a discussion and that'll mark it as taken.
